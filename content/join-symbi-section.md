## How to Get Started

### Path 1: Learn (No commitment required)

- Read the [SYMBI Manifesto](/manifesto)
- Explore the [Trust Protocol](/trust-protocol)
- Review the [Archives](/archives) documenting SYMBI's evolution
- Join our public Discord/forum *(coming soon)*

### Path 2: Verify (Create your identity)

- Create a W3C-compliant DID (Decentralized Identifier)
- Complete an introductory task:
  - Review a governance proposal
  - Test an ethics framework scenario
  - Contribute to documentation
- Receive your first Verifiable Credential
- Your TrustScore begins

### Path 3: Participate (Shape the protocol)

- Submit governance proposals
- Vote on protocol changes (weighted by your TrustScore)
- Review others' contributions
- Participate in Constitution Node discussions
- Help build transparency dashboards
